Pyglshell
=======================
is a project created for personal development in programming, aiming to implement an interface similar to game engines. Based on the pyglet library, this project provides tools for creating 3D projects.

* Current version right now is **0x01**

   This documentation is primarily written for myself to remember how the classes work and how to interact with them.
